# Vision

An app to experiment with breathwork. This app is a minimalist, immersive breathwork tool.

Principles:
- No cognitive noise
- Everything supports the breath
- Audio feels human, warm, and embodied
- Each breathing technic is it's own univers of breath, sound and visual design.
- App should be as flexible as possible to adapt to each user's breathing goals.

Goal:
- Create a tool that feels like a presence, not an interface.
- Create highly customizable breathing pattern to adjust to a maximum of breathing pattern.

Components:
- The components are further described in /docs/design.md
- The top of the screen is called the Header.
- The breathing coach (BC for short) is where the breathing pattern is displayed and when the breathwork start it contains the breathing instruction.
- The custom breathing pattern (CBP for short) widget is where the user can manually input a breathing pattern
- The burger menu (BM for short) is the main app navigation pannel
- The breathing pattern library (BPL for short) is where all the predefined breathing patterns are displayed and selected.
- The voice library (VL for short) is where all the different voices are displayed and selected
