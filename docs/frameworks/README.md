# Framework References

This folder stores framework guidance written by framework authors, plus small project indexes that explain how the app uses those references.

## Files

- [Chakra UI v3 full guide](chakraui-llms-full.txt) - raw upstream Chakra UI documentation.
- [Chakra UI project guide](chakraui.md) - project-specific map to the Chakra concepts used by this app.

## Rules

- Do not edit raw upstream framework files to add app-specific opinions.
- Add project-specific usage notes in a separate file next to the raw reference.
- Keep design decisions in `docs/design/`.
