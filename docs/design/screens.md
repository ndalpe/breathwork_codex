# Screens

## Idle Home

- Shows the header.
- Shows the breathing coach below the header.
- Shows the `Breathing pattern library` button below the breathing coach.
- Shows the `Custom breathing pattern` button below the breathing coach.

## Active Breathing Session

- Shows the header.
- Gives the breathing coach the available screen real estate.
- Hides the breathing pattern library entry point.
- Hides the custom breathing pattern entry point.
- Keeps the stop button visible and within the viewport.
- Keeps countdown digits fully visible above the stop button.

## Burger Menu

- Takes the full screen.
- Shows the centered `Menu` title.
- Shows the close button at the top right.
- Shows links to the breathing pattern library and voice library.
- Uses consistent visual styling for all menu buttons.
