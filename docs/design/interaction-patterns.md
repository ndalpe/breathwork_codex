# Interaction Patterns

## Breathing Pattern Library Button

- When no breathwork is running, display a button below the breathing coach so the user can access the breathing pattern library.
- This button should be labeled `Breathing pattern library`.

## Custom Breathing Pattern

- When no breathwork is running, display a button labeled `Custom breathing pattern`.
- Tapping this button expands the custom breathing pattern widget.

## Burger Menu

- When tapped, the burger menu should take all screen space.
- The burger menu should have a background color.
- The title of the burger menu is `Menu` and is centered.
- The title should use a contrasting color and be easy to read.
- The burger menu contains a link to the breathing pattern library.
- The burger menu contains a link to the voice library.
- A close button is visible at all times at the top right of the menu.
- The close button allows the user to close the burger menu.
- Buttons in the burger menu have a bold title followed by a line break and a description.
- The visual design of all burger menu buttons should be identical.

## Session State

- Starting a breathwork session hides the breathing pattern library entry point.
- Starting a breathwork session hides the custom breathing pattern entry point.
- Stopping a breathwork session returns the user to the idle controls.
