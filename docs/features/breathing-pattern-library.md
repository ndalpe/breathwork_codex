# Breathing Pattern Library

## Goal

Provide a library of predefined breathing pattern.

## Current Implementation
The library contains the most commun breathing patterns. Each pattern is described by a short explanation of the purpose of the breathing pattern.

The library is accessed through the burger menu. It should be the first item in the BM.

The button to access the BPL is labeled as "Breathing patterns" followed by a short sentence describing what the library is. For example: Choose a predefined breathing pattern according to how you want to feel.

Tapping on this button brings the BPL full screen.

Each predefined breathing pattern should have a name, a short description and the breathing pattern in seconds.

- Name: The name of the breathing pattern.
- Short description: The purpose of the breathing pattern.
- Breathing pattern in seconds: Only the number of seconds for each state. For example: 4-4-4-4 or 4-7-8-0

Tapping on a breathing pattern closes the BPL and set the BC with the selected pattern. Ready to start the breathwork.
