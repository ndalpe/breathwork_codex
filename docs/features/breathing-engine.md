# Breathing Engine

## Goal

Provide a simple, reliable breathing timer that lets the user experiment with custom and predefined breathwork patterns.

The engine should feel calm, precise, and invisible during practice.

## Current Implementation
Have the possibility to set a custom breathing pattern. Design a widget to manually enter time, in seconds, individually for inhale - hold after inhale - exhale - hold after exhale. The custom breathing pattern should be displayed in the burger menu below the Breathing pattern and the voice.

Have preset patterns for the most commun breathing patterns along with a one sentence description of each of them so I can choose a breathing pattern according to how I feel on the moment. The description of each pattern should explain the end result of the breathwork. We can choose the breathing from the burger menu.

The breath counterThe app should display, always visible, the number of seconds remaining for each steps. Bonus: have the app dictate the countdown of seconds, with a mute button on the top right of the screen. The remaining number of sec9onds should be displayed in big digits center middle of the screen.

The app UI:
Darkgray background, light grey numbers and buttons.
