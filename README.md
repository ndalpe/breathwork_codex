# breathwork_codex

## project purpose
Design a flexible breathwork app that can adapt to breathing technique, current emotional state and desired emotional state after the breathwork session

## Key features
- Predefined breathing patterns library
- Possibility to input custom breathing patterns
- Predefined voices for breathing instruction
- User interface meet WCAG accessibility standard

## Tech stack
- As light and lean as possible
- ReactJS, Vite, and Chakra UI

## Development

Requires Node.js 20 or newer.

```sh
npm install
npm run dev
```

Create a production build with `npm run build`. GitHub Actions publishes the generated `dist/` directory to GitHub Pages.
